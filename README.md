# React AG Grid - 회사 데이터 조회

사진 기반 회사 데이터를 AG Grid로 조회·검색하는 React 앱입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 으로 접속합니다.

## 기능

- 15건 회사 데이터 그리드 표시 (지역, 업종, 협력사, 담당자, 회사이름, a~eGroup)
- 전체 검색 (Quick Filter)
- 컬럼별 필터 (Floating Filter)
- 정렬, 페이지네이션
- 연두색 헤더 스타일 (원본 이미지와 유사)
